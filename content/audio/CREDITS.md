# Audio credits

Drop CC0 files here and the game uses them automatically; any missing
file falls back to the built-in synth (docs/03-CONTENT-DATA section 11).

- Music: `content/audio/music/<id>.ogg` for ids: title, hearth,
  hearthvale, westwood, ashenreach, northhollow, battle, boss, ending.
- SFX files are optional for v1.0; the synth covers every effect id.

Record attribution for every sourced file below, even for CC0.
